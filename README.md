# SUNLIGHT: Procurement Fraud Detection System

**Mission:** Make corruption impossible to hide through transparent, multi-source, expert-validated fraud detection.

> "We should make sure our platform when it is seen or used by any economist, tech, government official, law officer, they should be able to see 'ok this is a fraud' because of this and that with everything open like an open book, easy to see but with deep effectiveness."
