# SUNLIGHT: Government Procurement Fraud Detection
## 3-Month Bulletproof Build Plan

**Mission:** Build an academically-validated, legally-grounded AI system that detects procurement corruption with undeniable evidence.

**Timeline:** January 19 - April 19, 2026
**Launch Target:** April 2026
**Builder:** Rim (Burkina Faso → USA, F-1 visa, college student)

---

## CORE PRINCIPLES

1. **Academic Rigor:** Every claim backed by statistical evidence
2. **Legal Grounding:** Detection methods align with fraud prosecution standards
3. **Full Transparency:** All methodology, data, and code open source
4. **Expert Validation:** University professors review before launch
5. **Measured Claims:** "Fraud indicators" not "proven fraud"

---

By April 19, 2026:
- ✅ 10,000+ contracts analyzed with statistical rigor

- ✅ Academic paper submitted/published

- ✅ Credibility to approach governments/journalists/Elon

## RISK MITIGATION

**Risk:** APIs break/data unavailable
**Mitigation:** Multiple data sources, cached data, manual fallback

**Risk:** Professors decline to validate
**Mitigation:** Start outreach early, multiple targets, offer co-authorship

**Risk:** Legal pushback on claims
**Mitigation:** Measured language, cite precedent, lawyer review

**Risk:** Technical challenges exceed timeline
**Mitigation:** Weekly checkpoints, scope adjustment, focus on core features
